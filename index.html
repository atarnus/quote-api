Quotes from Gay Romance Novels API

Single quote record has following information:
  • Id
  • Author
  • Associated work (novel)
  • Associated series
  • The quote
  • Associated characters
URLcontent
/quotelists quotes with all details
/quote/#lists a quote with certain id number
/randomlists a random quote with all details
/authorlists all authors in database
/worklists all works in database
/serieslists all series in database
